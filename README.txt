War of the Sheep

Local play:
1. Install Node.js if you do not already have it.
2. Open Terminal in this folder.
3. Run: npm start
4. Open http://127.0.0.1:4173/

Public online multiplayer:
This must run on a real web server for friends outside your computer.
The easiest path is Render:

1. Put the project in GitHub.
2. Create a new Render Web Service.
3. Connect the GitHub repository.
4. Start command: npm start
5. Deploy.
6. Share the public Render link with your friends.

Important:
127.0.0.1 only works on the computer running the server.
Friends need the public hosting link, not the local link.
